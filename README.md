![costs](https://user-images.githubusercontent.com/108704306/209760812-982a8eec-117b-4c54-b779-041b9fdae551.png)


### Baixar dependências localmente

Após baixar o repositório:
```bash
- npm install
```

### Rodar o projeto

Executar o banco de dados:

```bash
- npm run backend
```

Executar o projeto:

```bash
- npm start
```
